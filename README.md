# Next.js Project

This project was bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

### 1. Install dependencies:
```bash
npm install
```
### 2. Start the Server:
```
npm run dev
```
# or
```
yarn dev
```
# or
```
pnpm dev
```
# or
```
bun dev
```
Then, open http://localhost:3000 in your browser.
